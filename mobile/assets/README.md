# Assets

Place the following files here before running EAS Build:

## Required images
- `icon.png` — 1024×1024 app icon (no transparency, used for iOS and Android)
- `adaptive-icon.png` — 1024×1024 foreground layer for Android adaptive icon
- `splash.png` — 1284×2778 splash screen image (or any tall portrait)
- `notification-icon.png` — 96×96 white-on-transparent Android notification icon

## Required fonts (in `assets/fonts/`)
Download from Google Fonts:
- `Sora-Regular.ttf`
- `Sora-SemiBold.ttf`
- `Sora-Bold.ttf`
- `Inter-Regular.ttf`
- `Inter-Medium.ttf`

Google Fonts links:
- Sora: https://fonts.google.com/specimen/Sora
- Inter: https://fonts.google.com/specimen/Inter

## Brand colors (for design reference)
- Leaf green: #6FAF7B
- Bloom coral: #F28C7B
- Sky cream: #FFF7F0 (splash/icon background)
- Deep moss: #294B45
